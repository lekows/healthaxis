# HealthAxis — MVP Visual

Plataforma web de saúde preventiva, pessoal e longitudinal.

## Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS (paleta personalizada)
- Lucide React (ícones)
- Recharts (gráficos de biomarcadores)
- Dados 100% mockados — sem backend ainda

## Rodar localmente

```bash
# 1. Instalar dependências
npm install

# 2. Iniciar servidor de desenvolvimento
npm run dev

# 3. Abrir no navegador
http://localhost:3000
```

## Rotas disponíveis

| Rota | Descrição |
|------|-----------|
| `/` | Landing page premium |
| `/dashboard` | Painel principal de saúde |
| `/exams` | Biomarcadores e exames |
| `/timeline` | Linha do tempo clínica |
| `/documents` | Documentos e laudos |
| `/anamnesis` | Questionário de saúde |
| `/report` | Relatório para consulta médica |
| `/profile` | Perfil de saúde pessoal |

## Aviso médico

O HealthAxis é uma plataforma de organização de dados de saúde.
Não realiza diagnósticos, não prescreve medicamentos e não substitui consulta médica.
